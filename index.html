PowerBI AutoBot

PowerBI AutoBot é um projeto de automação que desenvolvi para otimizar fluxos de trabalho no Power BI, simulando ações humanas através de reconhecimento de imagens e controle de interface. Utilizando Python com PyAutoGUI e OpenCV, a solução executa automaticamente atualizações de datasets e publicação de relatórios sem requerer licença Pro, permitindo atualizações programadas em múltiplos arquivos .pbix. Projetado para substituir processos manuais repetitivos, o bot demonstra minha capacidade de criar ferramentas práticas que resolvem problemas reais de negócios com eficiência. O código aberto destaca minha abordagem em automação inteligente e otimização de processos analíticos.

Algoritmo de Soma Consecutiva

Este projeto desenvolveu três algoritmos progressivamente otimizados para encontrar pares de números inteiros onde a soma dos antecessores iguala a soma dos consecutivos subsequentes. Partindo de uma abordagem força bruta, as soluções evoluíram utilizando otimizações matemáticas como a fórmula de Gauss e equações quadráticas, culminando em uma versão otimizada de tempo constante que descobriu mais de 15 pares com eficiência. As implementações demonstram melhorias significativas de desempenho, reduzindo o tempo de execução de minutos para resultados quase instantâneos através de refinamentos algorítmicos. O trabalho destaca habilidades de resolução de problemas e a capacidade de aprimorar eficiência computacional por meio de insights matemáticos, com metodologia completa e resultados comparativos documentados em formato de meta-artigo acadêmico.

AI SpeakerBot

AI SpeakerBot é um chatbot inteligente desenvolvido em Python que combina processamento de linguagem natural com capacidades de voz, utilizando as bibliotecas ChatterBot, pyttsx3 e SpeechRecognition. Space Shooter Rhythm

Space Shooter Rhythm é um jogo 2D em C++ que combina ação espacial com batidas musicais em tempo real. Utilizando OpenGL para gráficos e FMOD para sincronização sonora, o jogador controla uma nave, desvia de asteroides e atira em inimigos no ritmo da música. A jogabilidade é fluida, com efeitos visuais e sonoros que reagem à trilha sonora. Desenvolvido com foco em performance e interatividade, o projeto demonstra integração de sistemas gráficos e de áudio avançados. Ideal para quem aprecia jogos rítmicos com ação intensa.

Isometric Chess

Este projeto é um jogo de xadrez 2D desenvolvido em C++ utilizando OpenGL para renderização gráfica. O jogo permite movimentação e captura de peças, alternando entre dois jogadores e verificando a condição de vitória. A lógica de movimentação segue as regras tradicionais do xadrez, com peças como Peão, Cavalo, Torre, Bispo, Rainha e Rei. ServiceNow Ticket Generator

ServiceNow Ticket Generator é um sistema JavaFX desenvolvido para criação automatizada de chamados no ServiceNow via TABLE API. A aplicação oferece preenchimento inteligente de dados do usuário, templates configuráveis para diferentes tipos de tickets (Incidentes e NCRs), e integração direta com a plataforma ServiceNow. Com recursos como seleção de categorias, grupos de atribuição e status de chamados, o sistema simplifica e padroniza o processo de abertura de tickets. Desenvolvido com arquitetura MVC, armazenamento seguro de credenciais e geração de logs, é uma solução eficiente para equipes de suporte técnico.

Rhythm Sphere

Rhythm Sphere é um jogo musical desenvolvido em Unity que sincroniza a gameplay com a análise espectral de áudio em tempo real. Os jogadores interagem com esferas coloridas que caem em quatro pistas diferentes, cada uma correspondendo a padrões rítmicos específicos dos canais esquerdo e direito da música. O jogo utiliza algoritmos de processamento de áudio para gerar desafios dinâmicos, com sistema de pontuação baseado em precisão e combos. Implementa recursos como dificuldade adaptativa e salvamento de scores, oferecendo uma experiência rítmica imersiva. Desenvolvido com C#, emprega análise FFT e spawn procedural de objetos para criar uma harmonia perfeita entre jogabilidade e elementos musicais.
